# Citation_Recommendation
## Recommend papers system

### Data:

Open Research Corpus

### Make sure you have installed following libraries:

1. networkx 1.11
2. gensim
3. pymongo

### Run

`bash read_data.sh`

### MongoDB

`mongo -u admin -p admin123 --authenticationDatabase admin`
